# Ocean Portfolio - "Surface-to-Deep Dive"

An immersive ocean-themed portfolio that transforms as you scroll from bright aqua surface to deep navy depths.

## Features

- **Depth-Based Gradient**: Background smoothly transitions from surface to deep ocean as you scroll
- **3D Cursor-Following Shark**: Geometric low-poly shark companion that follows your cursor
- **Swimming Fish Projects**: Click on 3D fish to explore projects
- **Ocean Particle Effects**: Bubbles, fish schools, jellyfish, and light rays
- **Responsive Design**: Optimized for mobile with 2D fallbacks
- **Accessibility**: Full keyboard navigation and reduced motion support

## Tech Stack

- **Framework**: Next.js 14, React, TypeScript
- **3D Graphics**: React Three Fiber, Drei
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui

## Sections

1. Hero (Surface) - Bright aqua with ripple effects
2. Experience & Education (Shallow Reef) - Vibrant coral styling
3. Skills (Mid-Depth) - Pearl badges and bubble icon cloud
4. Projects (Reef) - Interactive 3D swimming fish
5. Hackathons (Deep) - Bioluminescent treasure aesthetic
6. Contact (The Abyss) - Darkest zone with glowing links
